# Sistem Miniatur Bendungan Pintar
Pada Posttest kali ini, kami mencoba untuk membuat perangkat IOT miniatur bendungan pintar untuk mendeteksi nilai air yang diintegrasikan dengan Kodular. Nantinya Kodular akan membantu kami untuk mengontrol dan memonitor perangkat melalui smartphone.

## Pembagian Tugas:
### Celio: Melakukan konfigurasi IOT melalui Arduino dan menyusun rangakaian block pada Kodular
### Rafif: Merakit rancangan IOT dan membuat tampilan pada Kodular

## Komponen yang Digunakan:
- Breadboard
- ESP32C3
- Kabel Jumper
- Buzzer
- Sensor Air
- Servo
